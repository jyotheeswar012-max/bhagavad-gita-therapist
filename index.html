· what it does ·

Ancient Wisdom, Modern Clarity

Share your emotion — get the exact shloka Krishna would have told you, translated and explained for today.

💛
12 Quick Emotion Buttons
Anxious, Heartbroken, Demotivated, Angry, Fearful, Lost, Self-Doubt, Distracted, Lonely, Jealous, Burnt Out, Rage — one tap to start.
📜
Smart Shloka Matcher
Groq AI reads your message and finds the most relevant verses from 100+ mapped emotion-to-shloka themes.
🤖
Groq AI Guidance
Warm, empathetic, non-preachy interpretations that connect 5,000-year-old wisdom to your exact situation.
🔊
Voice Recitation
Hear each verse spoken aloud via ElevenLabs, Edge TTS, or gTTS — a 3-layer fallback so audio always works.
📚
All 18 Chapters
Browse every chapter in the sidebar. Read shlokas with Sanskrit, transliteration and meaning at any time.
🔒
Zero Data Storage
No accounts, no logs. Inputs go to Groq for inference only. Session clears when you close the tab.

· setup ·

Run It Locally

Up and running in under 2 minutes.

# Clone the repo
git clone https://github.com/jyotheeswar012-max/bhagavad-gita-therapist
cd bhagavad-gita-therapist

# Install dependencies
pip install -r requirements.txt

# Add your Groq API key
cp .env.example .env

# Launch
streamlit run app.py

· tech stack ·

🐍 Python 3.11+ ⚡ Streamlit 🤖 Groq AI (LLaMA 3) 🔊 ElevenLabs / Edge TTS / gTTS 🖼️ Pillow
