# 🕉️ Bhagavad Gita AI Therapist

> *Share your struggles. Receive ancient wisdom. Find modern clarity.*

An AI-powered Streamlit app that listens to your emotional struggles and responds with relevant Bhagavad Gita shlokas + warm, practical guidance powered by **Google Gemini AI**.

[![Streamlit App](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://share.streamlit.io)

---

## ✨ Features

- 💛 **8 quick-emotion buttons** — Anxious, Heartbroken, Demotivated, Angry, Fearful, Lost, Self-doubt, Distracted
- 📜 **Smart Shloka Matcher** — Finds the most relevant shlokas for your situation from 50+ mapped themes
- 🤖 **Gemini AI Guidance** — Warm, empathetic, non-preachy modern interpretations
- 🎭 **Sanskrit + Transliteration + Meaning** — Full shloka display
- 🎨 **Beautiful dark UI** — Saffron & gold themed, inspired by Indian aesthetics
- ⚡ **Works without API key** — Graceful fallback responses built-in

---

## 🚀 Quick Start

```bash
git clone https://github.com/jyotheeswar012-max/bhagavad-gita-therapist
cd bhagavad-gita-therapist
pip install -r requirements.txt
cp .env.example .env
# Add your GEMINI_API_KEY to .env
streamlit run app.py
```

---

## 🔐 Get Your Free Gemini API Key

1. Go to [Google AI Studio](https://aistudio.google.com/)
2. Sign in → Click **Get API Key** → **Create API Key**
3. Paste it in the app sidebar or `.env` file
